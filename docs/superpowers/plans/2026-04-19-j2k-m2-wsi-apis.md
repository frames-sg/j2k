# J2K-M2 WSI APIs Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Architecture:** extend the existing backend adapter with resolution-aware construction, crop-after-decode ROI handling, reusable scratch buffers for rows, and a thin `J2kCodec` / `J2kContext` tile-batch layer.

**Tech Stack:** Rust, `slidecodec-core`, `slidecodec-j2k`, `dicom-toolkit-jpeg2000`
