# NVIDIA Baseline Artifacts

This directory stores CUDA validation artifacts used for comparison and release
evidence.

JSON and CSV artifacts are the source of truth. Do not hand-maintain benchmark
numbers in this README. Any public NVIDIA performance claim must cite the
recorded artifact, host, command, input source, comparator availability,
comparator version, and skipped paths.

Hosted CI is not NVIDIA performance evidence. Use self-hosted CUDA runners for
runtime and benchmark evidence.
